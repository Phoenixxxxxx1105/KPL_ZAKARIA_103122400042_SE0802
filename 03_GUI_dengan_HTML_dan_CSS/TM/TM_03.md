# Tugas Mandiri 03: Pemrograman JavaScript

## Soal

Setelah kamu menyelesaikan tugas pendahuluan (bisa buka di atas), terapkanlah fungsi untuk (1) menghitung huruf kecil yang disediakan di #hk, (2) mengubah huruf kecil ke huruf besar ketika pengguna menekan tombol #huruf-besar, dan (3) mengubah huruf besar ke huruf kecil ketika pengguna menekan tombol #huruf-kecil.
Untuk nomor 2 dan 3, tampilkan hasilnya di dalam editor-kecil.
Kemudian, hapuslah fitur "Paragrafkan" dari alat.

## Kode sumber

Tersedia di index.html, index.css dan index.js

## Deskripsi Program

Program ini adalah sebuah aplikasi web statis yang dirancang menggunakan tiga pilar utama pengembangan web: HTML untuk kerangka struktur, CSS untuk desain dan tata letak, serta JavaScript untuk memberikan interaktivitas (event-driven programming) melalui manipulasi DOM (Document Object Model).
Secara spesifik, penambahan JavaScript pada program ini berfungsi untuk menganalisis teks yang diketik oleh pengguna secara langsung (real-time).
